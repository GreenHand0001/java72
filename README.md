# Springboot Vue课程管理系统 java72

## 项目概述

本项目是基于Springboot和Vue的前后端分离的课程管理系统，适用于教务管理，实现教务管理员、教师、学生三方的功能需求。

## 技术栈

- 后端：SpringBoot
- 数据库操作：Mybatis-Plus
- 前端：Vue
- 数据库：MySQL

## 功能模块

### 系统角色

- 教务管理员
- 教师
- 学生

不同角色登录后，将看到不同的功能菜单。

### 功能列表

#### 教务管理员角色

- 管理员首页
- 系管理
- 专业管理
- 班级管理
- 学生管理
- 教师管理
- 课程管理
- 选课修改
- 管理员管理

#### 教师角色

- 教师首页
- 授课查询
- 教师课表
- 成绩录入

#### 学生角色

- 学生首页
- 选修课程
- 学生课程
- 课表查询
- 成绩查询
- 信息维护

## 运行环境

- 后端：Java 1.8+，SpringBoot框架
- 前端：Vue.js，Node.js环境
- 数据库：MySQL 5.7+

## 部署步骤

（此处具体部署步骤根据项目实际需求填写）

1. 数据库环境搭建，导入数据库脚本
2. 后端服务部署，配置相关参数
3. 前端项目构建，配置代理指向后端服务地址
4. 启动前后端服务，进行测试
